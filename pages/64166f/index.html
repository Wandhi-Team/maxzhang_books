修改运行中容器的挂载路径

要修改已经运行的 Docker 容器的挂载路径，需要先停止该容器，然后重新启动它并指定新的挂载路径。

# 停止容器

使用 docker stop 命令停止容器：

docker stop <container_name>

其中 <container_name> 是容器的名称或 ID。

# 修改挂载路径

使用 docker run 命令重新启动容器，并指定新的挂载路径。

如果使用的是 -v 选项进行挂载，则只需要修改该选项中的路径即可。

例如，假设原来启动容器时使用的命令是：

docker run -v /path/on/host:/path/in/container myimage

现在要将 /path/on/host 修改为 /new/path/on/host，则可以使用以下命令：

docker run -v /new/path/on/host:/path/in/container myimage

# 启动容器

使用 docker start 命令启动容器：

docker start <container_name>

容器会以新的挂载路径重新启动。
