软连接与硬连接

# 概念

Linux 链接分两种，一种被称为硬链接（Hard Link），另一种被称为符号链接（Symbolic Link）。

默认情况下，ln 命令产生硬链接。

# 语法

# 硬链接

源文件删除不会受影响(相当于源文件的备份)

ln 目标地址 本地地址
# 创建本地地址链接到目标地址
1
2

# 软链接

源文件删除会受影响(相当于源文件的投影)

ln -s 目标地址 本地地址
# 创建本地地址链接到目标地址
