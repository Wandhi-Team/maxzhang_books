linux用户及用户组

# 用户及用户组操作

# 用户

# 查看所有用户

cat /etc/passwd

# 查看所有用户组

cat /etc/group

# 查看当前用户的用户组

groups user

# 添加用户到用户组

将一个用户添加到用户组中，千万不能直接用：

usermod -G groupA

这样做会使你离开其他用户组，仅仅做为 这个用户组 groupA 的成员。 应该用 加上 -a 选项： usermod -a -G groupA user

(FC4: usermod -G groupA,groupB,groupC user) -a 代表 append， 也就是 将自己添加到 用户组groupA 中，而不必离开 其他用户组。

命令的所有的选项，及其含义：

# Options:
