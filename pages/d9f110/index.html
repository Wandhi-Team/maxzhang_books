linux下生成SSH key

系统：Centos7.4

# 生成SSH Key

设置用户名和email

git config --global user.name "username"

git config --global user.email "xxxx@qq.com"
1

# 生成ssh key

ssh-keygen -t rsa -C "xxxx@qq.com"
1

复制里面的信息，粘贴到git的ssh key 中
